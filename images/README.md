# Put your images here

**Portrait photo →** save it in this folder as exactly:

```
images/portrait.jpg
```

- Portrait orientation (taller than wide, roughly 4:5) looks best.
- If the file is missing, the site shows a striped placeholder instead — nothing breaks.
- If your photo is a `.png`, either rename it to `portrait.jpg` (the name matters, the
  actual format can still be PNG) **or** open `index.html`, find `images/portrait.jpg`
  in the HERO section and change it to `images/portrait.png`.

You can add any other images you want to link from your projects or posts here too.
